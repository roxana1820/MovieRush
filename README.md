# MovieRush - Full Stack Movie Application

MovieRush is a full-stack web application that allows users to securely register and log in to their accounts, explore trending movies, view detailed information, and manage their personal favourites list.

## Live Demo
https://movierush-0dl7.onrender.com


##  Features
### Authentication
Secure Registration: User account creation with encrypted passwords  
Login/Logout System: Session-based authentication to keep user data private

### Movie Discovery
Browse trending and popular movies  
Search movies by title  
Discover movies from different genres  

### Movie Details
View detailed information about each movie  
Integration with TMDB API for trailers, cast, ratings, and summaries  

### Favourites System
Add/Remove movies to personal favourites list  
View a personal list of favourite movies  

### Responsive Design
Fully optimized for mobile and desktop devices  

## Technologies Used

- CSS3

### Backend

- Express.js
